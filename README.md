# 地下城探索卡牌游戏

一款基于浏览器的 Roguelike 卡牌地牢爬塔游戏。

## 在线游玩

**https://dungeon-card-game.pages.dev**

## 开源说明

本项目为开源项目，欢迎学习与二次开发。

## 游戏玩法

- 使用 WASD 或点击地图移动
- 消耗饱食度探索地牢
- 遭遇敌人时进入回合制卡牌战斗
- 击败敌人获取金币与奖励
- 到达右下角出口进入下一层

## 技术栈

- 纯原生 JavaScript（无框架依赖）
- Canvas 渲染地图与角色
- 模块化架构：UI 管理、卡组系统、战斗系统

## 本地运行

直接用浏览器打开 `index.html` 即可，无需构建步骤。
