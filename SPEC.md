# 地下城探索卡牌游戏 - 规格文档

## 1. 项目概述
- **类型**: 回合制卡牌地下城爬行游戏
- **视觉风格**: 暗黑奇幻 + 金色点缀，带有神秘符文氛围
- **核心玩法**: 在随机生成的地牢地图中探索、打牌、击败敌人、收集奖励

## 2. 游戏机制

### 体力机制 (3种)
| 属性 | 范围 | 效果 |
|------|------|------|
| 生命值 (HP) | 0-100 | 降至0则游戏结束 |
| 能量 (Energy) | 0-3 | 每回合恢复1点，用于打牌 |
| 饱食度 (Food) | 0-100 | 每移动一步消耗5点，为0时每步损失10HP |

### 卡牌类型 (3种)
- **攻击牌**: 对敌人造成伤害
- **防御牌**: 获得护盾或减免伤害
- **技能牌**: 特殊效果（恢复、增益、减益等）

### 事件类型 (3种)
- **战斗事件**: 遭遇敌人
- **奖励事件**: 获得卡牌/金币/恢复
- **危险事件**: 陷阱/诅咒/突发事件

## 6. 状态机设计

### 状态枚举
| 状态 | 描述 | 允许的动作 |
|------|------|-----------|
| `start` | 开始界面 | start |
| `exploring` | 探索地图 | move, drawCards, endTurn, viewDeck |
| `battle` | 战斗中 | useCard, endTurn, flee, drawCards |
| `reward` | 获得奖励 | confirm, continue |
| `danger` | 遭遇危险 | confirm, continue |
| `level_complete` | 关卡完成 | confirm, continue |
| `game_over` | 游戏结束 | restart |

### 状态转换规则
```
start → exploring
exploring → battle/reward/danger/level_complete/game_over
reward → exploring
danger → exploring/game_over
battle → exploring/game_over
level_complete → exploring
game_over → start
```

### 各状态可用性检查
- `canMove()` - 仅在 exploring 可用
- `canUseCard()` - 仅在 battle 可用
- `canEndTurn()` - exploring 或 battle 可用
- `canFlee()` - 仅在 battle 可用
- `canDrawCards()` - exploring 或 battle 可用
